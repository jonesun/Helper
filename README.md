Helper
----------
实用工具-帮手，包含当前城市天气、日历、便签、物流查询、手机管理及手电筒、3d相册等功能。简单发布，欢迎试用，后续继续更新.

##Download

[v3.1.0 helper.apk](https://github.com/jonesun/Helper/blob/master/apk/helper.apk?raw=true)

**Compile environment**

```
compileSdkVersion=23
buildToolsVersion=24.0.0 rc1

minSdkVersion=14
targetSdkVersion=23
```
